# AI-Assisted Resume Analyzer & Job Recommender

A full-stack web application that analyzes resumes against job roles
using AI-assisted scoring logic and recommends suitable job profiles.

## What AI-Assisted Means
This project uses NLP-inspired keyword extraction and weighted scoring
to evaluate resume-job relevance in an explainable way.
